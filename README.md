# rm-calc

筋トレの重量と回数から推定MAXを算出し、各回数に対応するトレーニング重量を表示するプログラムです。

## 背景

筋トレにおいて、トレーニング重量の選択は非常に重要な要素である。
重すぎても軽すぎても、狙ったトレーニング効果を得ることは出来ない。

そこで、現在の自身の推定MAXや、
そこから逆算した回数ごとの推定MAX重量を
得られると便利である。

## 使用例

```bash
cargo run -- -t bp 100 5
```

- `-t bp`: ベンチプレス
- `-t sq`: スクワット

## 出力例

```text
 1   112.50
 2   107.14
 3   104.65
 4   102.27
 5   100.00
```
